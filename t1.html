Tema 1.Anàlisi i interpretació de fragmentació de codi

LES DADES

Les dades son representacions d'aspectes del mon real i són utlitzades per crear de noves o per prendre deicsions. EX: Una bases de dades pot recollir la informació de una plataforma de streaming, com Netflix i saber informació com: el nombre de usuaris, el historial de visualitzacions, les sèries o pel·licules més vistes...I amb aquesta es pot recopilar contingut persoanlitzat segons el gust del usuari.

Depenent de a qui van dirigides... Humans Programes o màquines
Estructura Tenen una estrucutra específica i comprensible. Les dades estna estruturades per ser processades per les màquines.
Formats Taules, gràfics, informes. JSON, XML o CSV.
Exemple Una taula de resultat de una competició nacional. Els sistemes ho processen automàticament sense intervenció humana.
